# 🎮 Online 3D Soccer Game

실시간 온라인 멀티플레이어 3D 축구 게임입니다.

## 📋 기능

### 🎯 핵심 기능
- **멀티플레이어**: 여러 플레이어가 동시에 플레이 가능
- **실시간 동기화**: WebSocket을 통한 60 TPS 게임 루프
- **물리 엔진**: 중력, 마찰, 반발 계수 구현
- **충돌 감지**: 플레이어-볼 자동 충돌 계산
- **팀 시스템**: 플레이어가 자동으로 팀 배치

### 🎨 그래픽
- 3D 그래픽 (Three.js)
- 조명과 그림자
- 실시간 플레이어 업데이트
- 필드 경계선 표시
- 플레이어 ID 라벨

### 📊 UI/HUD
- 실시간 플레이어 위치 정보
- 볼 좌표 표시
- 플레이어 목록 및 팀 색상
- 조작 가이드

## 🚀 설치 및 실행

### 필요한 환경
- Node.js 14 이상

### 설치
```bash
npm install
```

### 서버 실행
```bash
npm start
```

서버는 `http://localhost:3000` 에서 실행됩니다.

### 클라이언트 접속
브라우저에서 `http://localhost:3000` 접속

또는 여러 탭/창에서 접속하여 멀티플레이 테스트

## 🎮 게임 메커니즘

### 서버 (server.js)
- **GameState 클래스**: 중앙 집중식 게임 상태 관리
- **물리 시뮬레이션**: 중력(9.8 m/s²), 마찰력, 반발 계수(0.7)
- **충돌 감지**: 플레이어(거리 1.5m) - 볼 충돌 시 자동 킥
- **60 TPS 루프**: 모든 클라이언트에 60fps로 상태 브로드캐스트

### 클라이언트 (index.html)
- **Three.js 렌더링**: 조명, 그림자, 3D 모델
- **실시간 동기화**: 서버로부터 받은 상태로 즉시 업데이트
- **카메라 시스템**: 플레이어 추종 + 마우스 회전 제어

## 📊 게임 파라미터

### 물리 상수
- 중력: 9.8 m/s²
- 마찰력: 0.98 (매 프레임 적용)
- 반발 계수: 0.7 (바닥 바운스)
- 킥 파워: 25 m/s

### 필드 크기
- 가로: 50m (±25m)
- 세로: 30m (±15m)

### 게임 객체
- **플레이어**: 1×2×1 박스, 팀별 색상 (파란색/빨간색)
- **볼**: 0.5m 반지름 구체

## 🔧 기술 스택

- **백엔드**: Node.js + WebSocket (ws)
- **프론트엔드**: HTML5 + Three.js
- **통신 프로토콜**: JSON over WebSocket

## 📝 메시지 프로토콜

### 클라이언트 → 서버
```json
{ "type": "move", "x": 10, "z": 5 }
{ "type": "kick" }
```

### 서버 → 클라이언트
```json
{
  "type": "gameState",
  "time": 123,
  "players": [
    { "id": 1, "x": 10, "z": 5, "team": 0 }
  ],
  "ball": { "x": 0, "y": 0.5, "z": 0, "vx": 0, "vy": 0, "vz": 0 }
}
```
